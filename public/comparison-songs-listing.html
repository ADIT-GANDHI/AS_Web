🎯 Priority Fix List — Songs Listing Page

🔴 High Priority
1. Reduce marble texture height
The top section (wavy marble bg) is ~2× too tall. Cards must be visible above the fold on a standard 1080p monitor without scrolling.

2. Stop texture bleeding into card grid
Card grid background should be plain white/very light. The marble only belongs in the header/intro zone.
🟡 Medium Priority
3. Fix A–Z filter overflow
All 26 letters + "All" must fit in one horizontal row at 1280px. Reduce letter size/spacing or use a tighter font-size.

4. Card grid gap
Tighten the gap between cards to match the PDF's compact feel (~16px vs current ~24px).
